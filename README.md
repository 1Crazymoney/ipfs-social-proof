# ipfs-social-proof

Identity and social proof serverlessly via IPFS

## Requirements

* js-ipfs (browser mode)
* node.js 10
* Chrome / Firefox

## Install

* git clone / fork this repo
* `npm install`

## Build Bundle

* `npm start`
* This will produce a bundle one can consume in a web/electron app at `bundle.js`
